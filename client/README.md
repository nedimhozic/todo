$TodoAppClient
========

Look how easy it is to use:

    git clone project
	Open cmd within server folder
	 - run 'npm install http-server'
	 - run 'http-server -p 5000'

Features
--------

- You can create user, validate him and login
- You can create todo tasks and get them
